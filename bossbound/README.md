# Bossbound Coming Soon

One-page Netlify website met groot Bossbound-logo en Coming Soon tekst.

## Netlify instellingen

Build command:

```text
npm run build
```

Publish directory:

```text
dist
```

## GitHub structuur

Zet alle bestanden direct in de hoofdmap van je repository:

```text
index.html
package.json
netlify.toml
_redirects
css/
assets/
```
